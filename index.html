Connect Your Phone

# Step 1: Install Node.js (if not installed)
pkg update && pkg install nodejs -y

# Step 2: Download connector
wget https://your-url.com/connect.js

# Step 3: Connect!
node connect.js

Run these commands in Termux on your Android phone. Your phone will appear in the list once connected.
