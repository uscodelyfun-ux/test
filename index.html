üöÄ Getting Started

  1. Connect your phone: Install Termux on an old Android device
  2. Run the connector: Execute the setup script in Termux
  3. Create an API: Set up your first API endpoint
  4. Add data: Create collections and documents in the database
  5. Use your API: Make requests from any application

üì± Phone Setup

Step 1: Install Termux

Download Termux from F-Droid (NOT Google Play):

https://f-droid.org/en/packages/com.termux/

Step 2: Run Setup

# Update packages
pkg update && pkg upgrade -y

# Install Node.js
pkg install nodejs -y

# Download connector
wget https://your-domain.com/connect.js

# Run it
node connect.js your@email.com

üíª API Examples

JavaScript / Fetch

// Create a document
fetch('https://domain.com/api/u/username/todos', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ title: 'Learn backend', done: false })
})
.then(res => res.json())
.then(data => console.log(data));

cURL

# Get all documents
curl https://domain.com/api/u/username/todos

# Create document
curl -X POST https://domain.com/api/u/username/todos \
  -H "Content-Type: application/json" \
  -d '{"title":"New todo","done":false}'
